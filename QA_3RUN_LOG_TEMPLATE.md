# QA 3-Run Log Template (Device × Game)

목표: HOLD 항목 해소를 위해 **기기별/게임별 3회 플레이(각 60초 이상)**를 동일 형식으로 기록합니다.

## 작성 규칙
- 각 기기(Device Slot A/B)에서 게임별 3회씩 기록
- `DurationSec`는 실제 플레이 시간(초)
- `Result`는 PASS / FAIL
- FAIL일 경우 `Issue`에 재현 절차를 짧게 기록
