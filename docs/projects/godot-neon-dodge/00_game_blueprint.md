# 00_game_blueprint — Godot Neon Dodge

## 1) 게임 한 줄 정의
- 장르: 탑다운 회피 액션 (2D)
- 핵심 재미: 짧은 순간의 판단(위치/대시)으로 생존 시간을 늘리는 긴장감
- 타겟 플레이타임: 1세션 3~8분

## 2) 성공 기준 (Done)
- [ ] 30초 내 조작/목표를 이해할 수 있다
- [ ] 5분 플레이에서 크래시/프리징이 없다
- [ ] 난이도 상승이 부드럽고 억울사가 낮다
- [ ] 입력 반응(이동/대시)이 즉각적이다

## 3) 핵심 루프
1. 이동/대시로 탄막 및 적 개체를 회피
2. 생존 시간/근접 회피 보너스로 점수 획득
3. 점진적 난이도 상승(스폰율/속도) 대응

## 4) 범위
### In Scope (v0.1)
- 플레이어 이동/대시
- 단일 적 타입 + 스폰 시스템
- 생존 타이머/점수/HUD
- 게임오버/재시작

### Out of Scope (v0.1)
- 다중 스테이지/보스
- 영구 성장 메타
- 온라인 리더보드

## 5) 리스크 & 대응
- 리스크: 초반 난이도 설계 실패 시 이탈
- 대응: 스폰 곡선 데이터 분리 + 플레이테스트 기반 조정
