# 01_gdd — Godot Neon Dodge

## 게임 개요
- 컨셉: 미니멀 아레나에서 생존 시간을 극대화하는 회피 게임
- 타겟: 짧고 집중도 높은 캐주얼/하드코어 혼합층
- 세션 길이: 3~8분

## 시스템 설계
- 조작: WASD/화살표 이동, Space 대시
- 승패: HP 0 또는 피격 누적 시 게임오버
- 난이도 곡선: 시간 경과 기반 스폰율/속도 증가
- 점수: 생존 시간 + 근접 회피 보너스

## 콘텐츠 설계
- 적: Chaser(기본 추적형)
- 오브젝트: Arena 경계, 위험 경고 이펙트(후순위)
- 확장 예정: Shooter/Dasher 적 타입

## UX/UI
- HUD: 생존 시간, 점수, 대시 쿨다운, HP
- 피드백: 피격 플래시, 대시 잔상, 게임오버 패널
- 사운드: 대시/피격/게임오버 SFX (v0.2)

## 밸런스 파라미터
- 별도 데이터 파일(리소스/상수 스크립트)로 분리 관리
  - spawn_interval_base
  - enemy_speed_base
  - dash_cooldown

## 기술 제약
- 플랫폼: PC(Windows) 우선
- 목표 성능: 60 FPS 기준

## 오픈 이슈
- 피격 판정 완충 시간(i-frame) 적용 강도
