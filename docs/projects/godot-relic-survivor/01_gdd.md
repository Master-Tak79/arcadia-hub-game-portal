# 01_gdd — Godot Relic Survivor

## 게임 개요
- 컨셉: 붕괴한 유적지에서 파도를 버티며 유물 빌드를 완성하는 생존 액션
- 타겟: 로그라이크 반복 플레이 선호층
- 세션 길이: 10~20분

## 시스템 설계
- 조작: 이동(WASD/방향키), 대시(Shift), 일시정지(Esc)
- 승패: HP 0 시 게임오버
- 난이도 곡선: 시간 기반 적 스폰률/강화, 10분 미니 보스
- 점수: 생존 시간 + 처치수 + 보스 보너스

## 콘텐츠 설계
- 적 타입:
  - Grunt: 추적형 기본 적
  - Dasher: 준비 후 돌진형
  - MiniBoss: 고체력 + 패턴 공격
- 업그레이드(초기 12종): 공격 속도/탄속/관통/범위/회복/방어 계열

## UX/UI
- HUD: HP, 레벨, EXP, 타이머, 처치수, 현재 빌드 효과
- 피드백: 피격 플래시, 레벨업 연출, 보스 경고 배너
- 사운드: 공격/피격/레벨업/보스 경고 (v0.2)

## 밸런스 파라미터
- `spawn_curve`, `enemy_stats`, `upgrade_values`를 데이터 파일 분리 관리

## 기술 제약
- 플랫폼: PC 우선
- 목표 성능: 60 FPS, 300개 적 동시 처리 시 급락 최소화

## 오픈 이슈
- 대시 무적 프레임 길이와 쿨다운 밸런스
- 초반(1~3분) 전투 템포 조정
