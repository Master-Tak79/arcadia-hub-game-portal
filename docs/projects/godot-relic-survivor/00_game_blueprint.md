# 00_game_blueprint — Godot Relic Survivor

## 1) 게임 한 줄 정의
- 장르: 탑다운 로그라이크 생존 액션 (2D)
- 핵심 재미: 매 레벨업마다 빌드를 고르며 점점 강해지는 성장 쾌감
- 타겟 플레이타임: 1세션 10~20분

## 2) 성공 기준 (Done)
- [ ] 30초 내 기본 조작/목표를 이해한다
- [ ] 10분 플레이에서 핵심 루프(이동-전투-레벨업)가 끊기지 않는다
- [ ] 레벨업 선택 3지선다가 실제 빌드 차이를 만든다
- [ ] 10분 기준 보스 1회전까지 난이도 곡선이 자연스럽다

## 3) 핵심 루프
1. 자동 공격과 이동으로 적 군집을 회피/처치
2. 경험치 수집 → 레벨업 3지선다로 빌드 강화
3. 웨이브 상승과 엘리트/보스 대응으로 생존 시간 확장

## 4) 범위
### In Scope (v0.1)
- 이동 + 자동 공격(기본 발사체)
- 적 2종(기본/돌진)
- 레벨업 선택 3지선다(업그레이드 16종, 복합 효과 포함)
- 웨이브 타이머 + 미니 보스 1종
- HUD + 일시정지 + 게임오버/재시작

### Out of Scope (v0.1)
- 메타 영구 성장
- 다캐릭터 시스템
- 온라인 리더보드

## 5) 리스크 & 대응
- 리스크: 업그레이드 체감 부족으로 반복 플레이 동기 약화
- 대응: 업그레이드 수치/효과를 데이터화하고 시너지 조합 우선 설계
