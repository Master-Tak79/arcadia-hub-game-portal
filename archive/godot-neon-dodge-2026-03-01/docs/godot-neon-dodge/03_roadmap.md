# 03_roadmap — Godot Neon Dodge

## M1 — Core Playable (v0.1)
- 목표: 플레이어 이동/대시 + 스폰 + 생존 점수 + 게임오버
- 완료 기준: 5분 플레이 안정성 확보, 핵심 루프 동작

## M2 — Feel & Balance (v0.2)
- 목표: 피드백(사운드/이펙트), 적 타입 1종 추가, 난이도 곡선 보정
- 완료 기준: 억울사 감소, 재도전 동기 강화

## M3 — Polish (v0.3)
- 목표: UX polish, 설정 메뉴, 리더보드 준비
- 완료 기준: 배포 가능한 품질 체크리스트 통과

## 백로그
- 리플레이 시스템
- 일간 챌린지
- 모드(하드/엔들리스)

## 리스크 추적
- 밸런스 파라미터 과도 하드코딩 방지
- 엔티티 수 증가 시 성능 저하 모니터링
