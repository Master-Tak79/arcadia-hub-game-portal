# ROADMAP_0.3.0 (Meteor Dodge)

## 목표
`0.3.0`은 "플레이 품질 + 재플레이성 + 운영성"을 만족하는 첫 안정 버전으로 정의합니다.

## 현재 기준 (0.3.7)
- 모바일 조작 안정화 (하단 버튼 고정, 입력 폴백)
- 시작 카운트다운/보호구간
- 점진 난이도 조정
- 게임오버 UX 개선
- Howler.js 기반 기본 SFX 적용
